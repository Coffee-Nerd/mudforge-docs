# Plugin Development

Learn how to create plugins for MudForge Web Client.

## Getting Started

Plugins are written in Lua and can:
- Create custom widgets (canvas, HTML, text)
- Draw graphics and text
- Respond to MUD output with triggers
- Create command aliases
- Set up timers for periodic actions
- Access GMCP and MSDP protocol data
- Communicate with other plugins

## Quick Example

```lua
settings = {
  name = "My First Plugin",
  version = "1.0.0",
  author = "Your Name",
  description = "A simple example plugin"
}

local myWidget = nil

function initialize()
  -- Create a canvas widget
  myWidget = createWidget({
    type = "canvas",
    title = "My Widget",
    x = 100,
    y = 100,
    width = 200,
    height = 150
  })

  -- Draw on it (canvas widgets are automatically active when created)
  clear("#1a1a2e")
  drawRect(10, 10, 180, 130, "#4a4a6a")
  drawText("Hello MudForge!", 20, 50, "#ffffff")

  echo("Plugin initialized!")
end

function shutdown()
  if myWidget then
    destroyWidget(myWidget)
  end
end
```
