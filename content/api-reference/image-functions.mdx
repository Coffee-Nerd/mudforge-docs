# Image Functions

Functions for loading and rendering images.

> This page will be populated with verified API documentation from the source code.

## Image Loading

- `loadImage()` - Load an image from URL
- `preloadImage()` - Preload an image (returns Promise)

## Image Drawing

- `drawImage()` - Draw an image with optional transforms
- `drawImageAlpha()` - Draw an image with alpha transparency
- `drawImagePart()` - Draw a cropped portion of an image
- `drawImageRotated()` - Draw a rotated image
