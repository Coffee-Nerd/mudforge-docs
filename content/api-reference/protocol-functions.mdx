# Protocol Functions (GMCP/MSDP)

Functions for accessing MUD protocol data.

> This page will be populated with verified API documentation from the source code.

## GMCP (Generic MUD Communication Protocol)

- `getGMCPData()` - Get GMCP data (all or specific package)
- `getAllGMCPData()` - Get all GMCP data
- `getCurrentRoom()` - Get current room GMCP data
- `onGMCPUpdate()` - Subscribe to GMCP updates

## MSDP (MUD Server Data Protocol)

- `getMSDPVariable()` - Get an MSDP variable
- `getAllMSDPVariables()` - Get all MSDP variables
- `onMSDPChange()` - Subscribe to MSDP changes (this session)
- `onMSDPChangeGlobal()` - Subscribe to MSDP changes (all sessions)
