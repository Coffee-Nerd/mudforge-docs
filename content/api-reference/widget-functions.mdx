# Widget Functions

Functions for creating and managing widgets.

> This page will be populated with verified API documentation from the source code.

## Widget Lifecycle

- `createWidget()` - Create a new widget
- `destroyWidget()` - Destroy a widget
- `showWidget()` - Show a hidden widget
- `hideWidget()` - Hide a visible widget

## Widget Manipulation

- `moveWidget()` - Move a widget to a new position
- `resizeWidget()` - Resize a widget
- `setWidgetZOrder()` - Set widget layer ordering

## Widget Properties

- `getWidgetProperty()` - Get a widget property
- `setWidgetProperty()` - Set a widget property
- `widgetInfo()` - Get widget information

## Widget Events

- `registerWidgetEvent()` - Register an event handler
- `unregisterWidgetEvent()` - Unregister an event handler

## Active Widget

- `setActiveWidget()` - Set the active widget for drawing
- `getActiveWidget()` - Get the currently active widget
