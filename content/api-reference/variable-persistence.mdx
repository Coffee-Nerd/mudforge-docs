# Variables & Persistence

Functions for storing and retrieving data.

> This page will be populated with verified API documentation from the source code.

## Plugin Variables

- `setVariable()` - Set a plugin variable
- `getVariable()` - Get a plugin variable
- `deleteVariable()` - Delete a variable
- `getAllVariables()` - Get all variables

## MUSHclient Compatibility

- `SetVariable()` - PascalCase alias
- `GetVariable()` - PascalCase alias

## Table Persistence

- `saveTable()` - Save a table to localStorage
- `loadTable()` - Load a table from localStorage
