## Engine Types

MudForge supports two plugin engine types:

### Next-Gen Engine (Recommended)
```lua
settings = {
  engineType = "next-gen"
}
```
- Clean global API without widget ID prefixes
- Uses `setActiveWidget()` to target widgets
- Better performance

### Legacy Engine
```lua
settings = {
  engineType = "legacy"  -- or omit for default
}
```
- Widget ID as first parameter in drawing functions
- MUSHclient-compatible API style
