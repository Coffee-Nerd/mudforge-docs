# Triggers, Aliases & Timers

Functions for automation: responding to MUD output, creating shortcuts, and timed actions.

> This page will be populated with verified API documentation from the source code.

## Triggers

- `addTrigger()` - Add a trigger
- `removeTrigger()` - Remove a trigger
- `enableTrigger()` - Enable a trigger
- `disableTrigger()` - Disable a trigger
- `findTriggerByName()` - Find a trigger by name
- `toggleTriggerByName()` - Toggle a trigger by name
- `toggleTriggerGroup()` - Toggle a trigger group

## Aliases

- `addAlias()` - Add an alias
- `removeAlias()` - Remove an alias
- `findAliasByName()` - Find an alias by name
- `toggleAliasByName()` - Toggle an alias by name

## Timers

- `addTimer()` - Add a timer
- `removeTimer()` - Remove a timer
- `findTimerByName()` - Find a timer by name
- `toggleTimerByName()` - Toggle a timer by name

## JavaScript-Style Timers

- `setTimeout()` - Set a timeout
- `clearTimeout()` - Clear a timeout
