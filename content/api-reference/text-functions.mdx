# Text Functions

Functions for rendering text, managing fonts, and processing ANSI colors.

> This page will be populated with verified API documentation from the source code.

## Text Drawing

- `drawText()` - Draw text at a position
- `drawTextWrapped()` - Draw text with word wrapping
- `drawColoredText()` - Draw ANSI-colored text
- `drawTextAligned()` - Draw text with alignment

## Font Management

- `setFont()` - Set the current font
- `measureText()` - Measure text dimensions

## ANSI Processing

- `parseAnsiText()` - Parse ANSI codes from text
- `stripAnsiCodes()` - Remove ANSI codes from text
