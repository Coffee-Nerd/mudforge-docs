# MUD Interaction

Functions for interacting with the MUD server and terminal.

> This page will be populated with verified API documentation from the source code.

## Sending Commands

- `send()` - Send a command to the MUD
- `Execute()` - Execute a command (MUSHclient alias)

## Terminal Output

- `echo()` - Echo text to the terminal
- `print()` - Print to plugin output
- `colourNote()` - Output colored text (MUSHclient style)
- `Note()` - Echo text (MUSHclient alias)

## Plugin Commands

- `registerCommand()` - Register a plugin command
