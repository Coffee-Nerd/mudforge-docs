# Drawing Functions

Functions for drawing shapes, gradients, and performing canvas operations.

> This page will be populated with verified API documentation from the source code.

## Basic Shapes

- `drawRect()` - Draw a rectangle
- `drawCircle()` - Draw a circle
- `drawLine()` - Draw a line
- `drawEllipse()` - Draw an ellipse
- `drawPolygon()` - Draw a polygon
- `drawArc()` - Draw an arc

## Advanced Shapes

- `drawRoundedRect()` - Draw a rounded rectangle
- `drawBezier()` - Draw a Bezier curve
- `drawPath()` - Draw using path commands

## Gradients

- `drawGradient()` - Draw a linear gradient
- `drawRadialGradient()` - Draw a radial gradient

## Canvas Operations

- `clear()` - Clear the canvas
- `save()` - Save canvas state
- `restore()` - Restore canvas state
- `translate()` - Translate the canvas
- `rotate()` - Rotate the canvas
- `scale()` - Scale the canvas
- `setClipRegion()` - Set clipping region
- `resetClipRegion()` - Reset clipping region
- `setBlendMode()` - Set blend mode
- `setGlobalAlpha()` - Set global alpha

## Batch Operations

- `startBatch()` - Start batching draw operations
- `endBatch()` - End batching and execute
