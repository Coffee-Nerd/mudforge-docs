# Group Widget

Displays your party/group members with health, mana, and movement gauges. Also shows enemies when in combat.

---

## Accessing Settings

- **Desktop**: Right-click anywhere on the widget
- **Mobile**: Tap the Settings button

---

## Display Options

| Option | Description |
|--------|-------------|
| Show MN | Show/hide mana gauges |
| Show MV | Show/hide movement gauges |
| Show Numbers | Display current/max values |
| Show Labels | Display gauge labels (HP, MN, MV) |
| Show Percentages | Display percentage values |

---

## Gauge Appearance

| Option | Description |
|--------|-------------|
| Overlay Text | Show text on top of gauges |
| Center Overlay | Center the text (or right-align) |
| Side by Side | Arrange gauges horizontally |
| Thin Gauges | Use minimal height gauges |
| Darker Background | Color the gauge background |

---

## Display Modes

| Mode | Description |
|------|-------------|
| Normal | Full member cards with all info |
| Compact | Reduced spacing |
| Ultra Compact | Minimal display |

---

## Gauge Colors

Gauges change color based on percentage:

| Level | Default Color | When |
|-------|---------------|------|
| Low | Red | 0-30% |
| Medium | Orange | 31-60% |
| High | Green | 61-90% |
| Full | Green | 91-100% |

You can customize the threshold values and colors in settings.

---

## Member Information

Each group member shows:
- Name and level
- HP gauge (always visible)
- MN gauge (if enabled)
- MV gauge (if enabled)

---

## Enemy Display

When your group is in combat, enemies appear in a separate section with red styling.

---

## Quest Status

If your MUD sends quest data, members show:

| Status | Meaning |
|--------|---------|
| Ready | Can take a quest |
| Xs left | Quest in progress |
| Xs cooldown | Waiting for cooldown |
| N/A | Quest unavailable |

---

## Data Source

The group widget gets data from GMCP:
- `Group.Members` - Party member info
- `Group.Enemies` - Combat enemies

If you don't see group data, check that:
1. You're in a group
2. Your MUD supports GMCP group data
3. GMCP is enabled in your MUD settings

---

## Testing Without Data

Enable "Use Fallback Data" in settings to see demo data for testing the display without being connected or grouped.
