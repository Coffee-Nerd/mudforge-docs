# Status Bar Widget

Displays your character's health, mana, and movement as visual gauges. Shows character info like name and level.

---

## Accessing Settings

Right-click anywhere on the status bar to open the settings menu.

---

## Display Options

| Option | Description |
|--------|-------------|
| Show Mana | Show/hide the mana gauge |
| Show Movement | Show/hide the movement gauge |
| Show Numbers | Display current/max values (e.g., "100/150") |
| Show Labels | Display gauge labels (HP, MN, MV) |
| Show Percentages | Display percentage (e.g., "67%") |
| Show Character Info | Display name, level, and status |

---

## Gauge Appearance

| Option | Description |
|--------|-------------|
| Overlay Text | Show text on top of the gauge |
| Center Overlay | Center the text (or right-align) |
| Darker Outlines | Add stronger text shadows for readability |
| Side by Side | Arrange gauges horizontally |
| Thin Gauges | Use minimal height gauges |
| Darker Background | Color the gauge background |

---

## Display Modes

| Mode | Description |
|------|-------------|
| Normal | Full gauges with all information |
| Compact | Reduced spacing, smaller display |
| Ultra Compact | Single-line, minimal display |

---

## Gauge Colors

Gauges change color based on your percentage:

| Level | Default Color | When |
|-------|---------------|------|
| Low | Red | 0-30% |
| Medium | Orange | 31-60% |
| High | Green | 61-90% |
| Full | Green | 91-100% |

### Customizing Thresholds

In the settings menu, you can adjust:
- The percentage value for each threshold
- The color for each threshold level

---

## Character Info

When enabled, displays:
- Character name
- Level
- Class (if available)
- Status effects (Resting, Leader, etc.)

---

## Data Sources

The status bar gets data from your MUD via:

- **GMCP** - Char.Vitals, Char.Base packages
- **MSDP** - HEALTH, MANA, MOVEMENT variables

If you don't see data, your MUD may need to enable GMCP/MSDP in your account settings.

---

## Mobile vs Desktop

**Desktop defaults:**
- All information visible
- Side-by-side gauges

**Mobile defaults:**
- Numbers hidden to save space
- Character info hidden
